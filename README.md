# Fiddler Everywhere 资源包

## 包含文件
1. **Fiddler Everywhere 7.5.0.exe**  
   Fiddler Everywhere（跨平台抓包工具）的官方安装包，版本为7.5.0。

2. **Fiddler Everywhere 破解补丁.exe**  
   用于对Fiddler Everywhere进行激活的补丁程序。

## 注意事项
- 破解补丁属于非官方工具，使用可能违反软件的用户协议，建议优先支持正版；
- 运行补丁前请关闭Fiddler Everywhere程序，按补丁提示操作；
- 本资源仅作技术交流使用，请勿用于商业用途。
